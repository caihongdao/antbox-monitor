# 矿机冷却系统监控平台 - UI设计规范

## 设计理念
- **大屏友好**: 适配4K/8K大屏幕，信息密度适中，视觉层次清晰
- **工业美学**: 结合工业监控系统的实用性与现代Web设计的简洁性
- **实时感知**: 通过动画、颜色、图表直观展示系统状态变化
- **操作高效**: 常用功能一键可达，减少操作深度

## 参考系统
1. **工业SCADA系统** (如WinCC、iFix)
2. **数据中心监控系统** (如DCIM)
3. **矿机管理软件** (如Antminer、Whatsminer后台)
4. **现代仪表盘设计** (如Grafana、Kibana)

## 组件规范

### 卡片 (Card)
```
┌─────────────────────┐
│ 标题                │
│ 副标题              │
├─────────────────────┤
│ 内容区域            │
│                     │
└─────────────────────┘
```
- **圆角**: 8px
- **内边距**: 20px
- **背景**: 白色，可选悬停效果
- **边框**: 1px solid var(--gray-3)

### 数据卡片 (Data Card)
```
┌─────────────────────┐
│ 标签                │
│ 数值 48.08 PH/s     │
│ 趋势 ↗ 2.3%         │
└─────────────────────┘
```
- **强调数值**: 大字号，粗体
- **趋势指示**: 颜色 + 箭头 + 百分比
- **背景**: 根据状态可选色块

### 按钮 (Button)
```css
/* 主按钮 */
.btn-primary {
  background: var(--primary-color);
  color: white;
  border-radius: 4px;
  padding: 8px 16px;
  border: none;
}

/* 危险操作 */
.btn-danger {
  background: var(--error-color);
  color: white;
}

/* 幽灵按钮 */
.btn-ghost {
  background: transparent;
  border: 1px solid var(--primary-color);
  color: var(--primary-color);
}
```

### 表格 (Table)
- **表头**: 背景色浅灰，字体加粗
- **行间隔**: 斑马纹或悬停高亮
- **操作列**: 按钮或链接
- **分页**: 底部居中对齐

### 图表 (Charts)
- **响应式**: 适配容器大小
- **交互**: 悬停显示数值，点击筛选
- **图例**: 可点击隐藏/显示数据系列
- **时间选择**: 预设范围 + 自定义

## 交互设计

### 1. 加载状态
- **骨架屏**: 数据加载时显示内容轮廓
- **进度条**: 长时间操作显示进度
- **加载动画**: 小范围操作使用旋转图标

### 2. 反馈机制
- **操作成功**: 右上角绿色提示框，3秒后消失
- **操作失败**: 红色提示框，显示错误详情
- **确认对话框**: 危险操作前二次确认
- **表单验证**: 实时验证，错误字段高亮

### 3. 动画效果
- **页面切换**: 淡入淡出，300ms
- **卡片悬停**: 轻微上浮 + 阴影加深
- **数据更新**: 数值变化时颜色闪烁后恢复正常
- **图表更新**: 平滑过渡动画

## 响应式设计

### 大屏幕 (≥1920px)
- **布局**: 多列，充分利用空间
- **字体**: 适当放大
- **图表**: 显示更多数据点

### 中等屏幕 (1200px-1919px)
- **布局**: 适中列数
- **字体**: 基准大小
- **图表**: 标准密度

### 小屏幕 (768px-1199px)
- **布局**: 单列为主
- **导航**: 折叠式菜单
- **图表**: 简化，重点数据

### 移动端 (<768px)
- **布局**: 垂直排列
- **操作**: 触摸友好，按钮加大
- **图表**: 简化或隐藏次要图表

## 原型实现建议

### 技术选型
- **前端框架**: Vue 3 + TypeScript 或 React + TypeScript
- **UI组件库**: Element Plus (Vue) 或 Ant Design (React)
- **图表库**: Apache ECharts (功能丰富) 或 Chart.js (轻量)
- **状态管理**: Pinia (Vue) 或 Redux Toolkit (React)
- **路由**: Vue Router 或 React Router
- **HTTP客户端**: Axios

### 开发步骤
1. **搭建开发环境**: 创建Vite/Webpack项目
2. **设计系统实现**: 定义CSS变量，创建组件库
3. **布局组件**: 实现页面框架、导航、侧边栏
4. **数据卡片组件**: 实现KPI卡片、状态卡片
5. **图表集成**: 封装ECharts组件
6. **API集成**: 连接后端数据
7. **页面开发**: 逐个实现页面
8. **交互优化**: 添加动画、反馈
9. **性能优化**: 代码分割、懒加载
10. **测试**: 单元测试、集成测试

### 性能优化
- **虚拟列表**: 矿机列表等大数据量使用虚拟滚动
- **图表优化**: 大数据量时降采样显示
- **代码分割**: 按页面分割代码，减少首屏加载
- **缓存策略**: API数据合理缓存
- **图片优化**: 视频流使用WebP格式
